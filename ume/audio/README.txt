収録音声(子どもの声)の置き場所

■ ファイル名の規則
- reveal-0.m4a または reveal-0.mp3 : 0番デモの正体明かし音声
  (m4a優先で探すので、iPhoneのボイスメモで録ったファイルをリネームするだけでOK)
- 本実装(ステップ3以降)では reveal-1 〜 reveal-22 を同じ規則で追加予定

■ 台本(0番)
「ドクターフィッシュが つれたよ!
  おんせんで ひとの ふるい かわを たべてくれる おさかなだよ!」

■ 録音のコツ
- 静かな部屋で、口から15cmくらい離して録る
- 1ファイル=1セリフ。言い直しがあれば録り直す方が編集不要でラク
- iPhoneのボイスメモ → 共有でPCに送る → このフォルダに置いて git push
  (またはGitHubのWeb画面で ume/audio を開き Add file → Upload files でも可)

■ 注意
- ファイルが無い間は「※こえは じゅんびちゅう」と表示されます(合成音声は使いません)
