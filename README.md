# みつけた！みずのなかま

水のいきもの図鑑PWA。3〜5歳児のお風呂での発見体験(氷に閉じ込めた生き物フィギュア21体×毎晩1体)の相棒アプリ。

- 公開URL: https://mt44kr-eng.github.io/ofuro-sakana-zukan/
- デプロイ: `git push` でGitHub Actionsが自動デプロイ(約1分)

## 実装状況

- ✅ 技術検証: 照合=MobileNet確定(下記)、LINE内ブラウザで表示・音声OK
- ✅ フェーズ1: 基盤(マスタデータ・ストレージ3系統・図鑑シェル)
- ✅ フェーズ2: 0番デモ(氷割り→みつけた！)・品質証明・LINE導線(梅のみ表示)
- ✅ フェーズ3: パスワード解錠・親モード・キャプチャ(儀式→撮影→照合→みつけた！)
- ✅ フェーズ4: ゾーン完了/グランド完了「コンプリート！」演出(松は21体後にひみつリビール)
- ⬜ フェーズ5: PDF(印刷CSS)・設定(BGM/音量)・リセット・エクスポート/インポート

## 解錠パスワード(【仮決め】印刷前に正式決定)

- 竹(通常版): `SAKANA21` / 松(ひみつ版): `HIMITSU22`
- ⚙ボタン長押し(1.2秒)で「おうちのかた」画面 → 入力(全角・小文字でも可)

## 照合の基準ベクトル

- `data/refs/{id}.json` に置くと自動判定が有効になる(作り方は `data/refs/README.txt`)
- 無い生き物は「おうちのひとに みせてね」→ 親の「あっているよ」で登録(手動フォールバック)

## 確定済みの主要判断

- **照合**: MobileNet特徴ベクトル+コサイン類似度、しきい値0.79
  (実測: 同じ魚0.91〜0.96 / 違う魚0.59〜0.67。色ヒストグラムは背景支配で不成立)
  撮影画像は全体+中央クロップの2通りで照合し高い方を採用
- **ティア**: 梅(無料デモ)/竹¥3,980(21体)/松¥4,480(+ひみつの22体目)。解錠は竹用・松用の共有パスワード(同梱カード印字)
- **今日の生き物**: 進捗ベース(登録済みの次の番号が「今日」)
- **音声**: 収録した子どもの声のみ(TTS不使用)。梅/竹に22体目の枠は出さない。リセットは収集進捗のみ初期化。完了表記は「コンプリート！」のみ
- **PDF**: 印刷用CSS方式(依存ゼロ、iOSは共有→PDF保存)

ume/                旧・梅tierモック(LINE検証で使用済み。フェーズ3完了後に削除予定)

## アセットの入れ方(コード変更不要)

- イラスト: `img/creature/{id}.png`(白背景or透過)。未収集シルエットは自動で黒塗り生成
- 音声: `audio/creature/{id}.m4a または .mp3`(iPhoneボイスメモのm4aそのままでOK)
- まず `doctor_fish` の音声1本でパイプライン検証 → OK後に残りを一括投入
- idの一覧は `data/creatures.json`

## 開発メモ

- ストレージ: localStorage 3キー分離(`mizu-progress` / `mizu-settings` / `mizu-entitlements`)。リセットはprogressのみ削除
- 子ども向けUIに否定語(「失敗」「ちがう」)を使わない
- 撮影画像は端末内処理のみ。サーバー送信なし
- 案C(じぶんのさかな)はv2送り。今回は作らない
